# GESTION-EVENEMENTS

## Stack utilisée

- Next.js
- React
- Tailwind CSS
- JavaScript

## Description

Application frontend de gestion d’événements permettant :

- d’afficher les événements,
- rechercher un événement,
- consulter les détails,
- s’inscrire à un événement,
- créer un événement.

Les données sont simulées localement avec un fichier JavaScript.

## Installation

```bash
npm install
```

## Lancement du projet

```bash
npm run dev
```

## Fonctionnalités

- Liste des événements
- Recherche dynamique
- Détail événement
- Formulaire inscription
- Création événement
- Responsive design
- Loading state

## Choix techniques

Le projet a été réalisé avec Next.js afin de bénéficier d’un routage simple et d’une architecture React moderne.

Tailwind CSS a été utilisé pour accélérer le développement de l’interface utilisateur responsive.

Les données sont mockées localement pour se concentrer sur la partie frontend demandée.
